# User Validation Template (Level 5)

Use this file during real user testing on Stellar Testnet.

## Minimum Testers
- Target: 5+ unique testnet users
- Each user must complete:
  - Wallet connect
  - Create escrow
  - Release or refund flow
  - Feedback submission

## Google Form Fields (Required)
- Name
- Email
- Wallet address
- Role tested (buyer/seller/both)
- Rating (1-5)
- What worked well?
- What failed or felt confusing?
- Suggested improvements
